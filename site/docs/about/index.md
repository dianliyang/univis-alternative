# About

This site is a simpler public browser for CAU Kiel lecture data from UnivIS. It keeps the original structure, but makes it easier to move through it.

::: info Quick Rule
Use **Institutions** to answer **who offers a lecture**.

Use **Lectures** to answer **where a lecture sits in the lecture directory**.
:::

## Two Ways To Browse

There are two entry points because UnivIS organizes lectures in two different ways.

| Browser | Best for | Typical starting points |
| --- | --- | --- |
| **Institutions** | Who offers a lecture | faculties, institutes, centers, units |
| **Lectures** | Where a lecture belongs in the directory | study areas, degree programs, catalog categories |

## When To Use Which

Choose **Institutions** if you are following an organizational structure.

Choose **Lectures** if you are following a study-program or catalog structure.

Both paths can eventually lead to lecture list pages, but they answer different questions on the way there.

## Example

- Start with **Institutions** if you want to browse from a faculty or institute.
- Start with **Lectures** if you want to browse from a program area like a master's track or catalog section.

## Source

This site is built from public UnivIS data and links back to UnivIS whenever the original page matters.

## Disclaimer

This site is an independent interface for publicly available UnivIS data. It is not an official CAU Kiel or UnivIS website.

Content can be incomplete, outdated, or differ from the original UnivIS pages. When accuracy matters, always check the linked UnivIS source.
