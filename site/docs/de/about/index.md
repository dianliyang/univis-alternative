# Über

Diese Seite ist ein einfacherer öffentlicher Browser für Lehrdaten der CAU Kiel aus UnivIS. Die ursprüngliche Struktur bleibt erhalten, lässt sich hier aber direkter durchsuchen.

::: info Schnellregel
Nutze **Institutions**, um zu sehen, **welche Einrichtung eine Lehrveranstaltung anbietet**.

Nutze **Lectures**, um zu sehen, **wo eine Lehrveranstaltung im Lehrverzeichnis eingeordnet ist**.
:::

## Zwei Wege Durch Das Angebot

Es gibt zwei Einstiege, weil UnivIS Lehrveranstaltungen auf zwei verschiedene Arten ordnet.

| Browser | Gut für | Typische Einstiege |
| --- | --- | --- |
| **Institutions** | Welche Einrichtung etwas anbietet | Fakultäten, Institute, Zentren, Einheiten |
| **Lectures** | Wo etwas im Verzeichnis eingeordnet ist | Studienbereiche, Studiengänge, Verzeichnis-Kategorien |

## Wann Welche Ansicht Sinnvoll Ist

Wähle **Institutions**, wenn du einer organisatorischen Struktur folgst.

Wähle **Lectures**, wenn du einer Studiengangs- oder Verzeichnisstruktur folgst.

Beide Wege können am Ende zu Veranstaltungslisten führen, beantworten unterwegs aber unterschiedliche Fragen.

## Beispiel

- Starte mit **Institutions**, wenn du von einer Fakultät oder einem Institut aus gehst.
- Starte mit **Lectures**, wenn du von einem Studienbereich, Studiengang oder Katalogabschnitt aus gehst.

## Quelle

Diese Seite basiert auf öffentlichen UnivIS-Daten und verlinkt zurück auf UnivIS, wenn die Originalseite wichtig ist.

## Hinweis

Diese Seite ist eine unabhängige Oberfläche für öffentlich verfügbare UnivIS-Daten. Sie ist keine offizielle Website der CAU Kiel oder von UnivIS.

Inhalte können unvollständig, veraltet oder von den ursprünglichen UnivIS-Seiten abweichend sein. Wenn Genauigkeit wichtig ist, prüfe immer die verlinkte UnivIS-Quelle.
