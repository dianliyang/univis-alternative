# UnivIS Kiel Catalog

Static browser for public UnivIS course data from Kiel.

- Built from allowed pages only
- English-taught courses are easy to find with language badges and filters
- Every generated course page links back to the original source
- Semester scope is limited to the recent window from `SS 2024` through `SS 2026`
